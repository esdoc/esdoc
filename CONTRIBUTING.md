# Contributing

## Setup
```
git clone https://github.com/typhonjs-node-esdoc/esdoc.git
cd esdoc
npm install
```

## Running Tests
Just the tests:
```
npm run test
```

Runs test w/ coverage:
```
npm run test-coverage
```

## Install From Head
Install ESDoc from head revision.

```
git clone https://github.com/typhonjs-node-esdoc/esdoc.git
cd esdoc
npm install
npm run test
npm install -g ./
esdoc -h
```
